# Examples

## basic-comparison.sh
Compare two databases and generate full reports + migrations.

```bash
./basic-comparison.sh
```

## full-dump.sh
Dump a complete schema from a database for local cloning.

```bash
./full-dump.sh
```

## critical-only.sh
Show only breaking changes (type mismatches, nullability changes).

```bash
./critical-only.sh
```

**Note:** Edit the connection strings in each script before running!
